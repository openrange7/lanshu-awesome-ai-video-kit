★ 推荐

model-selector

15 模型购物顾问。覆盖商业 11 + 开源 4。"用哪个模型好" / "Sora 还是 Kling" / "哪个支持中文" / "哪个能本地部署" — 推荐 1-3 个模型 + 理由 + 对应 skill 链接。

触发:不知道用哪个
 ★ 数据驱动

prompt-translator

跨 11 商业模型转换器。基于 110 条人工对照基准的查表式转换,不靠 LLM 直觉。"把 Sora prompt 转成 Kling 写法" → 目标公式 prompt + 字段映射表 + 注意事项。

触发:已有 prompt 想换模型
🎯 模型专属 · Seedance / HappyHorse / Kling
Seedance
seedance-prompter
8 要素结构化提示词。来源:火山方舟 53 页 PDF。
触发:"帮我写 Seedance 提示词" / "做个视频"
 Seedance
seedance-storyboard
3-5 个分镜 + 4 维度组织。复杂剧情拆"镜头 1/2/3"。
触发:"把剧情拆成分镜" / "多镜头视频"
 Seedance
seedance-debugger
12 类诊断 + 修复版本。症状 → 根因 → 解法。
触发:"我的提示词出问题了" / "人脸不像了"
 HappyHorse
happyhorse-prompter
30-55 词紧凑 + 8s 节拍 + 原生音频路径(with X audible)。
触发:"5-8 秒紧凑短片" / "要原生环境音"
 Kling
kling-prompter
三套写法自适应(4 部分基础 / 5 层进阶 / 图生视频)。
触发:"可灵" / "图生视频" / "角色对话" / "中文剧情"
🌲 选 Skill 决策树
你想要什么? ├── 不知道用哪个模型(含 4 开源) → model-selector ★ ├── 已有 prompt 想换模型(Sora→Kling 等) → prompt-translator ★ ├── 中文剧情、原生音频+对话、图生视频 → kling-prompter ├── 5-8 秒紧凑 + 要原生环境音 → happyhorse-prompter ├── 10-15 秒单镜头复杂叙事 → seedance-prompter ├── 多镜头剧情切换 → seedance-storyboard └── 已有 Seedance 提示词但效果不对 → seedance-debugger

💡 还没单独写 skill 的模型(Sora/Veo/Runway/Pika/Hailuo/Hunyuan/Wan/即梦 + 4 开源),可以让 Claude 直接基于 methodology/ 里的公式手写(尤其是 13 商业速查 + 14 开源速查)。或者用 model-selector 帮你选最匹配的现有 skill。
