## TL;DR(60 秒看完)

**一句话**:**在图片上画一根红线作为飞行路径,然后用 Seedance 让 AI 沿着这条线生成无人机 FPV 镜头**。

**适用场景**:城市航拍 / 景区穿越 / 商业地产 / 楼宇宣传片 / 多地标大乱斗 / 文旅 / 短视频爆款

**为什么是 Seedance**:
- 严格遵循路径线的能力业界领先
- 单镜头 15s + 一镜到底
- 多地标空间连续性出色
- 红线/标记物擦除能力强(模型理解"参考标记 vs 最终成片"的区分)

## 3 步工作流

### Step 1:在图片上画红线,定义镜头路径

工具:任意画图软件(Photoshop / Procreate / iPad 画图 / PowerPoint 都行)

**4 个小注意点**:

1. **长度匹配时长** — 红线的长度/复杂度,要和你准备生成的视频时长匹配。线短 / 简单 → 5-8s 视频;线长 / 多转折 → 15s 视频。**避免线没飞完视频就结束**。
2. **画粗一点** — 红线画粗一点(8-15 像素),方便模型识别为"明显的引导标记"。太细模型可能当成街道线漏掉。
3. **转折清楚 + 收尾箭头** — 路径每个转折点都要明显,最终位置加一个小箭头,**明确镜头收尾方向**。
4. **复杂路线分段** — 如果路线特别复杂(>5 个转折),建议拆成两段或多段来做,后面再剪辑拼接(参考 Step 3)。

### Step 2:Seedance 提示词模板

把画好红线的图 + 下面这条 prompt 一起喂给 Seedance:

```
请擦除红线、箭头和所有辅助标记。红线和箭头仅作为镜头运动路径参考,
最终成片中不得出现。镜头以第一人称 FPV 视角呈现,超高速、电影级、
一镜到底,严格沿着图片中的红色路径运动,不要偏离、不要跳步、不要
简化路线。镜头从【起点】开始,经过【节点1】,随后【节点2】,再到
【节点3】,最后到达【终点主体】并完成【收尾动作】。画面要求超写实、
运动顺滑稳定、速度感强、空间连续清晰,不要重复建筑、不要变形、
不要文字、不要水印。
```

**把【】里的内容换成你自己的场景就行**。例如:
- 【起点】= "富士山山顶"
- 【节点1】= "贴近埃菲尔铁塔尖顶"
- 【终点主体】= "自由女神像火炬"
- 【收尾动作】= "绕火炬上升一周后定格"

### Step 3:后期剪辑(可选)

如果想做更强的节奏,可以把多段素材合并成 15s+ 的航拍宣传片。

4 个剪辑动作:
1. **选最顺的段落拼接** — 跑了 N 次取最稳的几段
2. **加 BGM** — 强化节奏感(电影感 / 史诗感 / Synthwave 都好用)
3. **基础调色** — 让天空、对比度、氛围统一
4. **加速 / 放慢** — 优化镜头节奏(开头慢、收尾快是常用结构)

## prompt 模板的 8 个关键设计

为什么这条 prompt 能稳定工作?拆解一下:

| 关键句 | 作用 |
|---|---|
| `请擦除红线、箭头和所有辅助标记` | **核心**:告诉模型"红线只是参考,成片不要有" |
| `仅作为镜头运动路径参考,最终成片中不得出现` | 加强约束,防止红线漏到成片 |
| `第一人称 FPV 视角` | 锁定相机类型(无人机/FPV 穿越机) |
| `超高速、电影级、一镜到底` | 三个风格锚点 |
| `严格沿着图片中的红色路径运动` | **核心**:绑定红线为路径 |
| `不要偏离、不要跳步、不要简化路线` | 强化"严格沿线"防止 AI 偷工减料 |
| `从【起点】开始,经过【节点1】,...` | timecode-style 路径节点叙事 |
| `不要重复建筑、不要变形、不要文字、不要水印` | 4 个标准 negative 约束 |

## 关键启示(本玩法的可迁移原则)

1. **"参考标记"与"成片"分离** — 模型可以理解"这个东西只是给你看路径用的,不要画到最终视频里"。这个原则可以推广到:
   - 在图上画区域框 → 让 AI 在框内加东西
   - 在图上贴文字提示 → 让 AI 按文字执行后擦除
   - 在图上标颜色 → 让 AI 按色块换颜色后擦除

2. **路径式 timecode** — 用"从 X → 经过 Y → 到 Z"的叙事代替时间戳 timecode,在视觉锚点明确时更稳。

3. **多重否定** — `不要偏离` + `不要跳步` + `不要简化` 三连否定,比单句更强力锁住 AI 行为。
